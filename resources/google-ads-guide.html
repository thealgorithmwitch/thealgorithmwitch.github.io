Step 1: Structural Definition

"Do not start inside Google Ads. Start with structure."

Required Inputs

  • Campaign name (normalized)
  • Goal (conversion action)
  • Landing page (final URL)
  • Audience + intent
  • Offer / Message

Output

A single clear campaign concept with one primary conversion action.

System Rule

"Campaigns exist to produce measurable outcomes, not just traffic."

Step 2: The Registry Protocol

UTM Builder Flow

source: google
medium: cpc
campaign: normalized_name
Optional: content (variation), term (keyword)

System Enforcements

Lowercase Normalization Clean Naming Consistent Variant Key

Prevents Failure

  • Fragmented Reporting
    Multiple versions of the same data point.
  • Unmatched Ads
    Crashes dashboard audit filters.

Step 4: Campaign Deployment

Setup Rules

  • Type: Search (default)
  • Goal: Match defined conversion
  • Bidding: Max Conversions / Manual CPC
  • Location: Must be strictly defined

Keyword Protocol

  • Group by intent, not themes
  • 1–5 keywords per group
  • Avoid broad match (unless intentional)
  • Remove low-quality legacy terms

Creative Rules

  • Match keyword intent exactly
  • Align with landing page message
  • Include clear, mystical action
  • Minimum 2-3 ads per group
Step 5: The Sacred Binding

Apply Tracking to Ads

Final URL Requirements

Include full UTM link string
Match Registry naming exactly
System Logic

"Ads data joins on campaign_name + landing_page."

Failure Case

If mismatched → Campaign becomes unattributed or split across reports.

Step 6: Launch Checklist (Hard Gate)

1. Tracking

  • UTM matches registry
  • GA4 receiving traffic
  • Conversion in DebugView

2. Ads Content

  • No disapproved ads
  • Keywords relevant
  • Budget confirmed

3. Data Integrity

  • Registry entry manual
  • GA4 source = google/cpc
  • Appears in Google Ads

Post-Launch Rituals

First 24–72 Hours

Impressions > 0
CTR > 0
Sessions match
Dashboard visible

Ongoing Optimization

Fix Immediately (Auto-Flagged)

  • Spend with 0 conversions
  • High Impressions + Low CTR
  • High Clicks + Low Site Conv

Naming Laws & Final Outcome

Non-Negotiable Rules

  • Consistent naming (no duplicates)
  • Match UTM campaign exactly
  • Avoid manual variations in Ads UI

Without this process: Attribution collapses into "direct", and dashboard insights become misleading.
