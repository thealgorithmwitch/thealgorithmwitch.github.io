Review pending jobs before they touch `jobs.json`.

This page is front-end only. It can inspect pending submissions, let you edit the fields, and simulate approve/reject outcomes locally. Final write access should be handled by Google Apps Script, GitHub Actions, or a small backend endpoint.

Queue sources
`pending-jobs.json` + local browser submissions stored at `algorithm-witch.pending-jobs`.
Final publish path notes:

1. Google Apps Script can receive approved jobs from this page and append them into a Sheet or JSON endpoint.
2. GitHub Actions can merge approved queue data into `jobs.json` and commit the change.
3. A small backend endpoint can write directly to `pending-jobs.json` or another datastore.
